El problema en San Ramón

Según encuestas municipales y reportes de organizaciones ambientales locales, en San Ramón se estima que entre el 60% y 70% de los residentes no conoce la ubicación exacta de los puntos limpios o contenedores de reciclaje.

Menos del 20% de los hogares separan sus desechos, lo que incrementa la contaminación por vertederos ilegales y satura los rellenos sanitarios.

Dato importante: El 65% de los residentes reciclarían si tuvieran herramientas sencillas para hacerlo.

Características de GreenLink

📍

Mapa Interactivo

Visualización de contenedores, centros de reciclaje y puntos limpios según tu ubicación.

🔍

Filtros Inteligentes

Búsqueda por tipo de material (plástico, vidrio, electrónicos, etc.) y horarios.

ℹ️

Información Detallada

Instrucciones de reciclaje, datos de contacto y contribución ecológica de cada centro.

👥

Comunidad Sostenible

Los usuarios pueden añadir nuevos puntos y calificar los existentes.

📱

Accesible

Plataforma web responsive que funciona en móviles y computadores.

⚙️

Tecnología IoT

Prototipo con sensores para monitorear el nivel de llenado de contenedores.

Tecnologías Utilizadas

Frontend

  • HTML5, CSS3, JavaScript
  • Bootstrap 5 (Diseño responsive)
  • Google Maps API o Leaflet.js

Backend

  • Firebase (Base de datos en tiempo real)
  • Autenticación de usuarios

IoT (Opcional)

  • Arduino UNO o ESP32
  • Sensores ultrasónicos HC-SR04
  • Plataforma Blynk o Firebase para datos
